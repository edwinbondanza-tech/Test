Git Cheat Sheet

Updated for January 2026 Standards

1. Setup & Config

Configure your identity and start new projects.

git config --global user.name "[name]" git config --global user.email "[email]" git init git clone [url]

2. Daily Workflow

Tracking changes and creating snapshots.

git status git add [file] git add . git commit -m "[message]" git commit --amend

3. Branching & Merging

Isolate work and combine features.

git branch git switch [branch-name] git switch -c [new-branch] git merge [branch] git branch -d [branch-name]

4. Synchronizing

Interact with GitHub, GitLab, or Bitbucket.

git remote add origin [url] git fetch [remote] git pull git push origin [branch]

5. Inspection & Recovery

Review history and fix mistakes.

git log git diff git stash / git stash pop git reset --hard [commit-id] git revert [commit-id]

Pro Tip: 2026 Syntax

Modern Git prefers switch and restore over the overloaded checkout command to improve clarity.

For more detailed guides, visit the Official Git Documentation.
